# Api

## Routes

### Auth ``/api/auth``

``/register`` :
- Create a new user in data base
- Body :
    ```json
    {
        "username": "exmaple",
        "email": "example@example.com",
        "password": "password"
    }
    ```
- Response :
    ```
    User created
    ```


``/login`` :
